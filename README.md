# Statistical Tester

Conducts statistical tests with simple sets of data OR parameters

## Description

Has:
1. z-tests
  a. One Proportion Z-Test
  b. Two Proportion Z-Test
2. t-tests
  a. One Mean T-Test
  b. Two Mean T-Test
3. chi-squared tests
  a. Goodness of Fit
  b. Homogeneity
  c. Independence

Writes out all parts of conducting the test
1. Name
2. Hypothesis
3. Conditions
4. Test Statistic and P-value
5. Conclusion
